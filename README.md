# えいごペット 🥚

子供向けの **英単語学習 × たまごっち風育成** Webアプリです。
英単語クイズに正解してキャラにごはんをあげ、毎日続けるほどキャラが育って進化します。

**▶ アプリを開く: https://gaogao999.github.io/eigo-pet/**

---

## 特徴

### 英単語クイズ
- 英検 **準2級・2級**（合計 約3,700語）から4択で出題
- **ふりがな付き**：日本語訳はすべてふりがな（ひらがな）表示。漢字が読めなくてもOK
- **音声読み上げ**：英単語を自然な英語ボイス（Samantha）で発音（🔊きく）
- **やさしいいみ**：日本語の選択肢を**長押し**すると、子供向けのかんたんな説明を表示
- **わからないボタン**：押すと答え（正解）を表示。その単語は復習にまわります

### 学習の仕組み（おぼえる → わすれない）
- **まちがえた単語が出やすい**重み付き出題（復習まち ×5、覚えた語は低頻度で再確認）
- **「おぼえた」判定**：新しい単語は1回正解、**一度まちがえた単語は2回連続で正解**すると「おぼえた」。途中で間違えるとやり直し
- **おぼえた しんちょく**を学習タブにバーで表示（おぼえた数 / 全単語数・ふくしゅうまち数）

### たまごっち風の育成（時間ベースのライフサイクル）
- 成長段階：**タマゴ → ベビー → キッズ → ヤング → アダルト**
- 各進化は「**時間の経過＋その日の勉強**」の両方で成立（勉強しないと進化しない）
  - ベビー1時間・キッズ1日・ヤング2日 → アダルトはおよそ3日目
- **寿命 10〜15日**：毎日の勉強・お世話で延び、放置・病気放置で縮む
- **毎日のお世話が必要**：おなかが空っぽのまま／病気のまま**約2日**ほうっておくと、寿命をまたずに**お別れ**になってしまう（早めにごはん・おくすりを。危なくなると警告が出ます）
- 寿命がくると **お別れ → 新しいタマゴで再スタート**（学習の進捗・着せ替えは引き継ぎ、育てた子は「けいふ」に記録）
- 各成長段階は **オリジナルのドット絵キャラ**（PNG）で表示。系譜は **タマゴ1種（たまご）→ ベビー1種（ぴよっこ）→ キッズ1種（ちびっこ）→ ヤング4種**（お世話ランク ⭐◎○△ に対応）**→ アダルト**。アダルトは **その世代（生まれてから）の もくひょうたっせい日数**でランク（⭐◎○△）が決まり、近いランクも一定確率で出る重み付き抽選。続ければ **全24種＋レア2種＝計26種**を図鑑にあつめられます
- ステータス：おなか・ごきげん・しつけ・体重
- **わがままサイン** → しつけボタンで対応
- **病気**：一定確率でかかる → **おくすり（えさ20）** で治療

### つづける しかけ
- **デイリー目標（1日20語）** とリング表示、達成でごほうび
- **連続記録（ストリーク）** ＋ 週カレンダー（月曜始まり）＋ **おやすみ券**（連続キープ）
- **週の宝箱**（5日達成で開く）／ **コンボ**（連続正解でえさ倍）／ **ごほうびタイム**（15〜17時は2倍）
- **称号（トロフィー）** ／ **効果音**（ON/OFF）

### ミニゲーム（あそぶ・えさ1）
- **ジャンプ**：障害物をよけて進む（2段ジャンプ可）
- **うみ**：自動でしずむキャラをタップで浮かせ、岩のすきまを通る

### おこづかい（家庭内）
- キャラと**お別れするとき**、余ったえさを**おうちのお金（バーツ）に買い取り**
- **1匹あたりの上限**とレート（えさ何個＝฿1）は**おうちの人が設定**（初期：えさ5＝฿1・上限฿200/匹）
- せってい→**おこづかい**で残高・履歴を確認、支払い後は残高リセット
- 実際の課金・決済は一切なし。**勉強を続けるほどおこづかいが貯まる**家庭内のごほうびの仕組み

### きせかえ・その他
- **はいけい（背景）**：おぼえた数でアンロックするドット絵の風景（やま・もり・うみ・ビーチ・さばく・とかい・がっこう など）
- **キャラの名前**：名前をタップして変更
- **データのバックアップ／復元**：せってい→データ から、コードで書き出し・読み込み
- **データは端末内に保存**：連続日数・覚えた数・キャラの成長が翌日も残る（端末ごと）

## ホーム画面への追加（iPhone）

1. **Safari** で https://gaogao999.github.io/eigo-pet/ を開く
2. 下の**共有ボタン**（□に↑）→ **「ホーム画面に追加」**
3. アイコンができ、タップすると全画面でアプリのように使えます

> Safariで開いてください（Chromeでは「ホーム画面に追加」が出ません）。

## 画面構成

下部タブ（左右スワイプでも切替）：

- **そだてる**：キャラ・ステータス・お世話ボタン
- **がくしゅう**：デイリー目標・進化予告・おぼえた進捗・べんきょう開始
- **せってい**：ずかん／きせかえ／けいふ／たんご（単語一覧）／データ

## 構成（ファイル）

| ファイル | 役割 |
|---|---|
| `index.html` | 画面テンプレート（Design Component）。表示時に React を読み込んで描画 |
| `support.js` | Design Component ランタイム（自動生成・編集しない） |
| `app.js` | アプリ本体のロジック（育成・クイズ・ゲーム・描画） |
| `words.js` | 英単語データ（英語・日本語・ふりがな・品詞）とバージョン番号 |
| `easy.js` | 「やさしいいみ」辞書 |
| `manifest.webmanifest` | PWA設定（ホーム画面追加用） |
| `icons/` | アプリアイコン |
| `characters/` | キャラクターのドット絵（36×36 PNG）。タマゴ・ベビー・キッズはオリジナル制作、ほか30種 |

> 表示時に React/ReactDOM を CDN（unpkg）から読み込むため、初回表示はネット接続が必要です。

## バージョン

ホーム画面の最下部に表示される `バージョン revXX` で、いま見ているバージョンを確認できます。
更新時は `words.js` 先頭の `APP_REV` を上げています。

## クレジット

- 英単語データ：[bangluutru/eiken-workbook](https://github.com/bangluutru/eiken-workbook)（出典 eiken-csv）
- ふりがな生成：[pykakasi](https://github.com/miurahr/pykakasi)

## ライセンス / 利用について

個人・家庭内での学習利用を目的とした非商用のプロジェクトです。
キャラクターのデザインはオリジナルで、既存キャラクターの複製は含みません。
